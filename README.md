# 🚚 BIDS - Logistics Bidding Platform

## 📌 Overview
BIDS is a React-based logistics management application that allows users to create, view, and manage transport bids. The application simulates a real-world bidding system used in logistics operations.

---

## ✨ Features

### 🔐 Authentication
- User Signup & Login
- Basic session management using localStorage
- Protected routes

### 📊 Dashboard
- View all live bids in a structured table
- Search bids by city
- Status badges (Live, Responded, Unresponded)

### ➕ Create Bid
- Form to create new bids
- Auto-generated Bid ID
- Data updates instantly in dashboard

### 📄 Bid Details
- View detailed information of each bid
- Route, vehicle, weight, and staff details

### 💾 Data Persistence
- All bids stored in localStorage
- Data remains after refresh

---

## 🛠 Tech Stack

- React JS (Hooks)
- React Router
- CSS (Custom styling)
- LocalStorage (State persistence)
